MyPlan Task Manager
===================

A console application that allows to store tasks.

Quick Start
-----------

Use the application as follows:

1. Create a JSON file for a data base:
   ```
   PS> "[]" > "my-plan.json"
2. Point the file on run
   ```
   PlanManager.exe path\to\my-plan.txt
   ```
3. Edit tasks and conclude with 'q' command
